argument-hint: "[CSV, JSON, or file path] [--no-docx] [--no-xlsx]"

Load the `vc-portfolio` skill and generate a portfolio report.
Exports markdown + DOCX + XLSX by default; use `--no-docx` for markdown only, `--no-xlsx` to skip XLSX.
Pass all arguments through to the skill.
